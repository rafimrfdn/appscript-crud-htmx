Express + Appscript + HTMX Crud App

Selamat datang di aplikasi input data sederhana.

Di sini anda bisa melakukan CRUD (create, read, update, dan delete) data sederhana.

Motivasinya: untuk belajar Javascript.

Tentang Aplikasi

Aplikasi ini berjalan secara serverless. Tidak perlu menyewa hosting untuk berjalan, cukup disimpan di Vercel saja.

Node akan menjalankan Express JS.

Lalu Express akan menjalankan sejumlah route untuk mengeksekusi tiap perintah CRUD.

Dengan bantuan HTMX kita bisa lebih mudah menuju ke route tertentu untuk mengeksekusi perintah di dalam server.

Kemudian data yang dikirim akan dihandle oleh Appscript.

Appscript sendiri memiliki sejumlah code untuk melakukan request CRUD.

Sehingga database bisa diakses/dimodifikasi melalui Google Sheet.

Kekurangan aplikasi ini

  • Server respon lambat (di atas 3 detik)
  • Kurang estetik
  • Memiliki banyak bug yang mesti diperbaiki
